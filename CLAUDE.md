## Project Overview

Solin-website is a minimal Python starter project. It currently contains a sample Python script (`main.py`) and a static HTML page (`hello_world.html`).

## Environment

- **Python 3.9** with a local virtual environment in `.venv/`
- **Black** formatter configured via PyCharm
- No external dependencies installed yet (no `requirements.txt`)

## Commands

```bash
# Activate virtual environment
source .venv/bin/activate

# Run the sample script
python main.py

# Format code with Black
black .
```
